# simple_calculator
CalcApp is a simple console-based calculator written in a Windows Batch file.  
It was created as a first practice project to learn how to work with batch scripting, user input, conditions, and basic file handling.

CalcApp is a simple console-based calculator written in a Batch file.  
It is my first practice project, created to learn the basics of batch scripting, user input handling, conditions, and simple file operations.

### Features
- Asks the user for their **name** and **ID**
- Menu with 4 basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Option to perform **all** operations
- Displays the result(s) in a clean and formatted layout
- Option to save the operation details to `operation_data.txt`, including:
  - User information (name & ID)
  - Input numbers
  - Selected operation(s)
  - Creation date and time
- Option to print the contents of the saved text file inside the console

### How to run
1. Download the `.bat` file.
2. Double-click it on Windows, or run it from Command Prompt (CMD).
